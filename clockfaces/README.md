# Clockfaces

Community and third-party Canvas clockface themes for [Clockwise](https://clockwise.page/).

These `.json` files are compatible with the **Canvas clockface** (cw-cf-0x07) and can be loaded at runtime — no reflashing required.

## How to use

1. Flash Clockwise firmware with the Canvas clockface included
2. Host these `.json` files on any web server (or use the [Clock Club local server](https://github.com/jnthas/clock-club))
3. In the Clockwise settings, set:
   - **Canvas Server** → your server URL
   - **Canvas File** → the filename (without `.json`)
